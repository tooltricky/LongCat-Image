这是一个非常棒的自媒体内容生产思路。利用代码自动化生成标准化的“快闪”视频素材，能极大提升你的做号效率。

为了满足你的需求，我编写了一个 **单文件 HTML 解决方案**。

### 这个工具的特点：
1.  **完全本地运行**：不需要服务器，双击打开即可。
2.  **自动录制**：点击开始后，程序会在 Canvas 上渲染动画，并自动在后台录制，播放结束后自动下载 `.webm` 格式视频（大多数剪辑软件和平台都支持）。
3.  **可视化坐标拾取**：为了解决“每张图文字位置不一样”的痛点，我增加了一个**调试模式**，鼠标在画面上移动即可看到当前的 X, Y 坐标，方便你填写配置。
4.  **视觉冲击力**：加入了“肯·伯恩斯效应”（缓慢推拉镜头）和动态绘制红圈的效果。

### 详细配置指南

#### 1. 替换素材
在代码大约第 70 行左右，你会看到 `config` 变量。
将 `src` 里面的链接替换为你电脑里的图片路径（建议图片放在和 html 同一个文件夹下，例如 `images/1.jpg`）或者直接使用网络图床链接。

**注意**：图片比例最好就是 16:9，否则会被裁剪放大。

```javascript
// 示例
src: './素材/电商海报.jpg', 
```

#### 2. 设置红圈位置（关键步骤）
由于每张图中文的位置不同，我做了一个辅助功能：
1.  运行网页。
2.  点击 **“开启/关闭 坐标拾取模式”**。
3.  鼠标移动到图片上你想圈出的文字的**左上角**，记下 X 和 Y。
4.  修改代码中的 `highlight` 数据：

```javascript
highlight: { 
    x: 200, // 左上角横坐标
    y: 300, // 左上角纵坐标
    w: 400, // 矩形宽度（大概估算一下，或者鼠标移动到右下角减去左上角）
    h: 100  // 矩形高度
}
```

### 生成效果预想
*   **0-3.3秒**：展示第一张图，画面缓慢放大，0.5秒时一个红色的框迅速绘制出来，圈住重点文字，红框带有发光效果。
*   **3.3-6.6秒**：瞬间切换到第二张图（快闪风格），重复上述动画。
*   **6.6-10秒**：切换到第三张图，重复动画。
*   **结束**：浏览器自动弹出一个下载框，保存为 `.webm` 视频。

### 后续剪辑建议
生成的视频是 `.webm` 格式，你可以直接导入剪映、必剪或 Premiere。导入后，建议配上快节奏的 **卡点音乐**（如“快闪”、“科技感”类BGM），配合音效（如“叮”的一声）在红圈出现时，效果会非常专业，完全符合你“AI工具猎手”干货、高效的人设。
