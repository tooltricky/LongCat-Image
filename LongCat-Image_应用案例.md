# LongCat-Image 应用案例集

## 核心优势总结

基于README.md的技术报告，LongCat-Image的最擅长的任务是：

### **中文文字渲染 + 照片级真实感图像生成**

**核心优势：**
1. **行业领先的中文文字渲染能力**：在ChineseWord基准测试中达到90.7%的覆盖率，远超竞品（HunyuanImage-3.0: 58.5%, Qwen-Image: 56.6%, Seedream 4.0: 49.3%）
2. **卓越的照片真实感**：在真实感(Realism)的人工评估中获得3.60分，超越所有对比模型
3. **高效的模型设计**：仅6B参数，在多个基准测试中超越参数量更大的开源模型
4. **双语支持**：中英文双语能力，适合国际化应用场景

**最适合的应用场景：**
需要在真实感照片中嵌入中文文字的所有场景，包括但不限于：商业广告设计、电商产品图、社交媒体内容创作、出版物配图、品牌视觉设计等。

## 提示词使用注意事项

根据README.md中的特别说明，使用LongCat-Image进行文字渲染时：

### ⚠️ 重要提示

**必须将目标文字用引号（""）包裹起来**

**原因：** Tokenizer对引号内的内容采用字符级编码。如果不使用引号，文字渲染质量会显著下降。

**正确示例：**
```python
prompt = '一个咖啡杯旁边的木牌上写着"慢生活"两个大字'
```

**错误示例：**
```python
prompt = '一个咖啡杯旁边的木牌上写着慢生活两个大字'  # 缺少引号，渲染效果会很差
```
